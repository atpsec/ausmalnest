# AusmalNest

German-first MVP for educational coloring pages.

## Scope

- Next.js App Router
- German SEO structure
- 10 prepared content categories
- First seed collection of printable coloring pages
- Print-friendly route for each page
- Sitemap and robots configuration
- No child accounts, uploads, comments, or licensed characters

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Product direction

AusmalNest targets parents, teachers, and educators, not direct child data collection. The initial model is free SEO traffic plus later premium PDF bundles.
